== Description ==

SB TBFA is a plugin that allows to load Twitter Bootstrap and Font Awesome on your WordPress site. You can manage option to turn on or turn of Bootstrap and Font Awesome easily.

**Required Plugin**

* [SB Core](https://wordpress.org/plugins/sb-core/)

**Current Versions**

* Bootstrap: 3.2.0
* Font Awesome: 4.2.0

**Translations**

* English
* Vietnamese

**Recommended WordPress Plugins**

* [SB TBFA](https://wordpress.org/plugins/sb-tbfa/)
* [SB Clean](https://wordpress.org/plugins/sb-clean/)
* [SB Comment](https://wordpress.org/plugins/sb-comment/)
* [SB Paginate](https://wordpress.org/plugins/sb-paginate/)

== Installation ==

Upload the SB TBFA plugin to your blog, activate it, then you can use Twitter Bootstrap and Font Awesome on your site.

1, 2, 3: You're done!
